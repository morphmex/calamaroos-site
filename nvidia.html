Installazione driver NVIDIA NVIDIA Drivers Installation

Guida passo-passo per installare i driver proprietari NVIDIA su CalamaroOS / Gentoo, con blacklist di Nouveau e configurazione di Xorg, dracut e GRUB. Step-by-step guide to install proprietary NVIDIA drivers on CalamaroOS / Gentoo, with Nouveau blacklisting and Xorg, dracut and GRUB configuration.

ℹ️ Prima di iniziare ℹ️ Before you start

Tutti i comandi vanno eseguiti come root (oppure preceduti da sudo). Assicurati di avere una connessione internet attiva e di aver effettuato un backup dei file di configurazione che andremo a modificare. La password di default è calamaroos. All commands must be run as root (or prefixed with sudo). Make sure you have an active internet connection and have backed up the configuration files we are going to modify. The default password is calamaroos.

1
Configurare make.conf Configure make.conf

Apri il file /etc/portage/make.conf e aggiungi le righe per accettare le licenze e dichiarare la scheda video NVIDIA. Open /etc/portage/make.conf and add the lines to accept licenses and declare the NVIDIA video card.

# nano /etc/portage/make.conf
ACCEPT_LICENSE="*" VIDEO_CARDS="nvidia"
2
Maschera versioni problematiche e attiva dist-kernel Mask problematic versions and enable dist-kernel

Alcune versioni dei driver possono non funzionare con specifiche schede video. Il flag dist-kernel fa sì che, ad ogni aggiornamento del kernel, i driver NVIDIA vengano reinstallati automaticamente. Some driver versions may not work with specific video cards. The dist-kernel flag ensures that NVIDIA drivers are automatically reinstalled at every kernel update.

# echo ">x11-drivers/nvidia-drivers-580.126.18" >> /etc/portage/package.mask/nvidia-drivers # echo "x11-drivers/nvidia-drivers dist-kernel" >> /etc/portage/package.use/nvidia-drivers
⚠️

Esempio reale: sulla GTX 1070 è stato necessario mascherare la versione 595 perché incompatibile. Adatta il numero di versione alla tua scheda se riscontri problemi. Real example: on the GTX 1070 it was necessary to mask version 595 due to incompatibility. Adjust the version number for your card if you encounter issues.

3
Installare i driver Install the drivers

Aggiorna il sistema con il flag world per scaricare e compilare i driver NVIDIA. Update the system with the world flag to download and compile the NVIDIA drivers.

# emerge -DNuag @world
4
Configurare Xorg Configure Xorg

Il comando nvidia-xconfig genera automaticamente la configurazione di Xorg per la scheda NVIDIA. The nvidia-xconfig command automatically generates the Xorg configuration for the NVIDIA card.

# nvidia-xconfig
5
Blacklist del driver Nouveau Blacklist the Nouveau driver

Disabilita il driver open-source Nouveau per evitare conflitti con i driver proprietari NVIDIA. Disable the open-source Nouveau driver to avoid conflicts with the proprietary NVIDIA drivers.

# nano /etc/modprobe.d/blacklist-nouveau.conf
blacklist nouveau options nouveau modeset=0
6
Configurare dracut Configure dracut

Configura dracut per escludere completamente Nouveau dall'initramfs e includere la blacklist. Configure dracut to completely exclude Nouveau from the initramfs and include the blacklist.

# nano /etc/dracut.conf.d/nouveau.conf
# Force nouveau and drm in the initramfs for NVIDIA cards (Kepler and later) #force_drivers+=" nouveau drm drm_kms_helper " # Disable Nouveau completely in the initramfs omit_dracutmodules+=" nouveau " install_optional_items+=" /etc/modprobe.d/blacklist-nouveau.conf " # Optional but recommended hostonly="yes"
7
Parametri kernel in GRUB Kernel parameters in GRUB

Apri /etc/default/grub e modifica la riga GRUB_CMDLINE_LINUX_DEFAULT aggiungendo i parametri per disabilitare Nouveau e attivare il KMS NVIDIA. Open /etc/default/grub and modify the GRUB_CMDLINE_LINUX_DEFAULT line by adding the parameters to disable Nouveau and enable NVIDIA KMS.

GRUB_CMDLINE_LINUX_DEFAULT="rw quiet splash nouveau.modeset=0 rd.driver.blacklist=nouveau nvidia-drm.modeset=1"
8
Rigenera initramfs e GRUB, riavvia Regenerate initramfs and GRUB, reboot

Rigenera l'initramfs con dracut, ricostruisci la configurazione di GRUB e riavvia il sistema. Regenerate the initramfs with dracut, rebuild the GRUB configuration and reboot the system.

# dracut -f # grub-mkconfig -o /boot/grub/grub.cfg # reboot
✅

Tutto pronto! Dopo il riavvio i driver NVIDIA proprietari saranno attivi. Verifica con nvidia-smi. All set! After reboot the proprietary NVIDIA drivers will be active. Verify with nvidia-smi.
